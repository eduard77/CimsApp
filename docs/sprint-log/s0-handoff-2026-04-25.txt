DEFAULTS ALREADY AGREED (no need to re-ask)
--------------------------------------------
- Token: 32-byte random, base64url-encoded, returned once.
- At rest: SHA-256 hash only; plaintext never persisted.
- Default expiry: 7 days. Maximum: 30 days. Bootstrap: 24h.
- Email bind: optional; if set, Register must use matching email.
- Single-use: yes. Consumed tokens stay in DB for audit.
- Bootstrap: first consumer of a bootstrap token auto-becomes
  OrgAdmin globally.
- Revocation: not this sprint (v1.1 backlog B-001).
- Email delivery: out of scope for S0 (OrgAdmin shares token
  out-of-band for now).
