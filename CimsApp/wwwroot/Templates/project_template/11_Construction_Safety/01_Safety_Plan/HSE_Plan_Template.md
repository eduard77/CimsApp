## 3. SITE SETUP & SECURITY

- Site boundary: hoarding to minimum 2.0m height
- Single controlled access point for pedestrians
- Separate vehicle access with banksman
- 24hr CCTV + out-of-hours security
- Welfare facilities (Schedule 2 CDM 2015):
  - WC + washing facilities
  - Drying room
  - Canteen with hot water
  - Rest area
  - First aid room

## 4. INDUCTION & COMPETENCY

### 4.1 Induction Requirements
- All site workers to receive project-specific induction before first day on site
- Induction records held in `/11_Construction_Safety/04_Inductions/`
- Minimum content:
  - Site rules
  - Emergency procedures
  - Site-specific hazards
  - PPE requirements
  - Accident reporting
  - Drug & alcohol policy

### 4.2 Competency
| Role | Minimum Requirement |
|---|---|
| All operatives | CSCS card (UK) / equivalent |
| Supervisors | SSSTS / SMSTS |
| Plant operators | CPCS / NPORS card |
| Scaffolders | CISRS card |
| Electrical work | ECA / NICEIC competent |
| First aiders | HSE First Aid at Work |

## 5. PERMITS TO WORK

| Activity | Permit Type |

| Hot works | Hot work permit |

| Excavation > 1.2m | Excavation permit |
| Roof access | Working at height permit |

## 8. INCIDENT REPORTING

### 8.1 Reporting Timeframes
| Severity | Notify PM Within | RIDDOR? |
|---|---|---|
| Fatal | Immediate | Yes — immediate |
| Major injury / specified injury | 2 hours | Yes — 10 days |
| Over-7-day injury | 24 hours | Yes |
| First aid | Same shift | No |
| Near miss | Same shift | No — investigate |
| Dangerous occurrence | 2 hours | Yes |
| Environmental spill | Immediate | Env Agency if reportable |

### 8.2 Investigation
All incidents investigated. Major incidents use root-cause analysis. Reports in `/11.../05_Incidents/`.
